
Usage:

  skice <command> [<path>] [options...]

Commands:

  help, -h, --help [command]  Outputs the help text.
  version, -v, --version      Outputs the version number.
  new                         Creates a new sketch project.
  run                         Starts a local server to serve the sketch project.
  upgrade                     Upgrades the project form an older version.
